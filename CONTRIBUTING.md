# Contributing

Thanks for your interest in contributing!

## How to Contribute

- **Fix outdated links** — Tools and frameworks move fast. If you find a dead link or outdated info, please submit a PR.
- **Add resources** — Found a great tutorial, paper, or tool? Add it to the relevant section with a one-line description.
- **Improve explanations** — If something is unclear or could use a better example, PRs are welcome.
- **Translate** — Chinese (中文) translations are especially welcome.

## Guidelines

- Keep descriptions concise (one line per resource)
- Include links to primary sources (official docs > blog posts > videos)
- Maintain the opinionated, practical tone — this is a guide, not an exhaustive list
- Test that all links work before submitting

## What We Don't Want

- Marketing content or promotional links
- Resources behind paywalls (unless exceptionally valuable)
- Duplicate entries
- AI-generated summaries without human review
